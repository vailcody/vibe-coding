04

Возможности

  • Быстрое прототипирование — от идеи до UI за минуты
  • Автоматизация рутины — парсинг, отчёты, скрипты
  • Внутренние инструменты без очереди к разработчикам
  • Генерация тестов, документации, код-ревью
  • Личная эффективность — анализ данных
05

Ограничения

  • Не заменяет разработчика на сложных архитектурных задачах
  • Garbage in → garbage out
  • Контекстное окно ограничено — декомпозиция
  • AI может генерить уязвимый код — ревью обязательно
  • Галлюцинации: выдумывает API и библиотеки
