Что такое вайб-кодинг?

Новый подход: описываешь что хочешь на естественном языке → ИИ-агент пишет код.

Вы не пишете код — вы описываете вайб, а ИИ его реализует. Андрей Карпаты, ex-OpenAI
Для разработчиков
Ускорение в 5-10x на рутинных задачах
Для не-разработчиков
Возможность создавать рабочие прототипы самостоятельно

Возможности

  • Быстрое прототипирование — от идеи до рабочего UI за минуты
  • Автоматизация рутины — парсинг, отчёты, скрипты
  • Внутренние инструменты — без очереди к разработчикам
  • Генерация тестов, документации, код-ревью
  • Личная эффективность — анализ данных, автоматизация документов

Ограничения

  • Не заменяет разработчика на сложных архитектурных задачах
  • Garbage in → garbage out: нужно понимать, что хочешь получить
  • Контекстное окно ограничено — большие проекты нужно декомпозировать
  • Безопасность: AI может генерить уязвимый код — ревью обязательно
  • Галлюцинации: иногда «выдумывает» API и библиотеки

Почему начинаем с браузерных агентов?

  • Низкий порог входа — не нужно настраивать окружение
  • Быстрый результат — вау-эффект за минуты
  • Встроенные UI-инструкции — красивый результат из коробки
  • Кодовая база = контекст для локального агента
  • Дешёвая проверка — протыкали, нашли corner cases
